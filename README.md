Mini Jarvis 🤖
Overview

Mini Jarvis is a simple AI-based virtual assistant that executes tasks using voice commands. It is designed to automate basic daily operations and provide a smooth human-computer interaction experience.

Features
Voice command recognition
Open applications
Web search functionality
Get current time and date
Basic system automation
Tech Stack
Python
SpeechRecognition
pyttsx3
OS / Webbrowser modules
Installation
Clone the repository
git clone https://github.com/your-username/mini-jarvis.git
Navigate to the project folder
cd mini-jarvis
Install dependencies
pip install -r requirements.txt
Usage

Run the script:

python main.py
